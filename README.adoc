= Scripts to Try Project Syn

This project contains scripts hosted at https://try.syn.tools to try Project Syn the easiest way possible.

== Requirements

* `kubectl`
* `minikube`
* `k3d`
* `jq`
* `curl`
* A GitLab account.

== How to Use

. Check requirements:
+
[source,bash]
--
curl -fsL https://try.syn.tools/requirements | bash
--

. Launch your Minikube cluster
+
[source,bash]
--
minikube start --kubernetes-version=v1.23.8
--

. Launch your K3s cluster
+
[source,bash]
--
k3d cluster create steward --image=rancher/k3s:v1.23.8-k3s1
--

. Install Lieutenant in Minikube:
+
[source,bash]
--
curl -fsL https://try.syn.tools | bash
--

. Install Steward in K3s script:
+
[source,bash]
--
curl -fsL https://try.syn.tools/steward | bash
--

== Cleanup

. Cleanup at the end:
+
[source,bash]
--
curl -fsL https://try.syn.tools/cleanup | bash
--
